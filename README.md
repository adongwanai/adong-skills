# adong-skills

A collection of Claude Skills for extending Claude Code capabilities.

## Skills

### andrej-karpathy-perspective
Andrej Karpathy's perspective on AI systems, LLM learning, software paradigm shifts, human-in-the-loop agent design, and from-scratch technical understanding. Includes a self-contained research bundle with writings, conversations, expression DNA, decisions, external views, and timeline notes.

See [andrej-karpathy-perspective/](./andrej-karpathy-perspective/) for details.

### github-to-skill
Search GitHub repositories and generate Claude Skills from them. Helps you discover high-quality open-source projects and quickly turn them into reusable skills.

See [github-to-skill/](./github-to-skill/) for details.

## Usage

1. Copy a skill directory to your Claude skills path:
   ```bash
   cp -r andrej-karpathy-perspective ~/.claude/skills/
   ```

2. Restart Claude Code or reload skills

3. Use the skill in Claude Code
